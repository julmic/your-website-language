# Site HTML Statique - Arkadhya

Ce dossier contient le site web statique du centre ayurvédique Arkadhya.

## Structure du dossier

```
site-html/
├── index.html              # Page d'accueil
├── contact.html            # Page de contact
├── a-propos.html           # À propos
├── services.html           # Vue d'ensemble des services
├── massages.html           # Liste des massages
├── cures.html              # Liste des cures
├── blog.html               # Index du blog
├── philosophie-vedique.html # Hub philosophie
├── consultations-ayurveda.html # Page consultations
├── atelier-cuisine.html    # Atelier cuisine
├── styles.css              # Styles CSS complet
├── script.js               # JavaScript vanilla
├── assets/
│   └── images/             # Images du site (à ajouter)
├── blog/
│   ├── les-3-doshas.html
│   ├── hiver-ayurveda.html
│   ├── sommeil-ayurveda.html
│   └── incompatibilites-alimentaires.html
└── legal/
    ├── mentions-legales.html
    ├── politique-confidentialite.html
    └── cgv.html
```

## Déploiement

Ce site est entièrement statique et peut être déployé sur :
- Hostinger
- OVH
- Infomaniak
- Tout hébergeur web classique

Il suffit de copier le contenu du dossier `site-html/` à la racine de votre hébergement.

## Technologies

- HTML5 sémantique
- CSS3 pur (pas de framework)
- JavaScript vanilla
- Google Fonts (Inter + Lora)
- Design responsive
- Support dark mode automatique
